My Project idea

Spatial-Temporal Analysis of the Effectiveness of GRACE Satellite in Monitoring Groundwater Storage in Medium-Scale Basins

This project aims to assess the effectiveness of the GRACE satellite in monitoring groundwater storage in medium-scale basins. Using GRACE satellite data, which provides measurements of the Earth's gravity field, we will estimate changes in groundwater storage over time. By analyzing the spatial and temporal patterns, we can evaluate the satellite's ability to monitor groundwater resources in these basins. This information will be valuable for water resource management and sustainable development in the studied regions.
